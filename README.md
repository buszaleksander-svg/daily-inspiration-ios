# Daily Inspiration – PIA14

Daily Inspiration is a simple iOS app created in SwiftUI as part of a group university project.

The app offers short inspirational quotes in a soft, minimal interface. Users can browse quotes, save favorites, and view quote history.

## Features

- Browse inspirational quotes
- Save favorite quotes
- View quote history
- Clean and minimal SwiftUI interface
